# Basic Sample Hardhat Project

Ensure to Create a ".env" file in the root folder with the following variables

    - CONTRACT_NAME
    - CONTRACT_SYMBOL
    - CONTRACT_ADDRESS
    - WALLET_OWNER 
    - WALLET_PRIVATE_KEY
    - POLYGONSCAN_KEY
    - ALCHEMY_KEY_TEST
    - ALCHEMY_KEY_PROD
    - JSON_CID
    - IMAGES_CID
